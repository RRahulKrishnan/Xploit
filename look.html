Meta-data Check Challenge
Using Exif Tool

Problem Statement:

In this challenge, you are tasked with analyzing the metadata of a suspicious PDF document to uncover a hidden flag. This exercise will test your ability to inspect files and detect information hidden within the metadata.



Scenario:




Objective:

  • Download the PDF: Download the provided PDF file and use the Exif.tools service to dig into its metadata.
  • Extract Metadata: Visit Exif.tools to inspect the metadata of the PDF document.
  • Submit the Flag: Once you uncover the flag in the metadata, submit it to complete the challenge.


Skills Required:

  • Metadata Analysis: Use tools to extract and examine the metadata of the document.
  • PDF Forensics: Investigate PDF files and uncover hidden or embedded information.
  • Basic File Analysis: Familiarity with file inspection tools for metadata extraction.


Hint:


When you open the Exif.tools page you will be greeted with this console where you can simply upload the file whoes meta data you would like to check.


After you click "Upload File" you will be redirected to a page which will show you the complete details and meta data of your file.




Conclusion:

This challenge tests your skills in extracting and analyzing metadata from files, a critical ability when investigating digital documents. By looking deeper into the structure of files, you can uncover hidden secrets and data that aren't immediately visible.
